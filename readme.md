# Retro Masters Golf ⛳

A first-person, N64-style golf game inspired by Augusta National.

### Features
- First-person swing view
- Ball-flight camera with shot tracer
- Physics-based shots
- Retro low-poly aesthetic
- Fully browser-based (GitHub Pages ready)

### Controls
- Click & Hold: Charge shot power
- Release: Swing

### Hosting
Just push to GitHub and enable Pages:
Settings → Pages → Deploy from main branch
